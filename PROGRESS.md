# 开发进度

> 最后更新：2026-04-07

---

## ✅ 已完成

### 基础框架
- Vite + React + Tailwind CSS 项目搭建
- 三栏布局：左侧导航（200px）+ 主内容区 + AI 教练（280px）
- Zustand 全局状态管理，localStorage 持久化
- Notion API 接口预留（USE_NOTION 开关）

### 知识场景
- **今日信息流**：粘贴全文/URL 自动解析（Claude API），文章卡片展示
- **内容添加弹窗**：已知 CORS 封锁域名识别，降级提示优化
- **存入知识库**：标签选择器（预设 + 自定义）+ 备注
- **导师解读**：将文章 + 学习目标发给 AI 教练触发 coaching
- **知识库 Tab**：标签筛选，移出，导出 Markdown（Blob 下载）
- **学习进度 Tab**：目标设定、统计数据、教练评估、周报告生成

### 产品场景
- **产品管理**：弹窗列表新建/切换/删除，空状态引导
- **维度评分**：5 级定性标签，inline 编辑名称和点评
- **社媒评测**：Claude API 生成结构化评测，加入分析/移除
- **文档上传**：拖拽/点击上传 txt/md，基于文档触发锐评

### AI 教练
- 流式输出，打字机光标，20 条截断，100 条历史上限
- 动态 System Prompt（注入学习目标、产品信息）
- 顶部学习进度卡片（知识场景）
- 场景切换通知（notice 消息）
- 自动阶段锐评（7 天触发一次，≤150 字）

### 周报告 & 导出
- 周报告：收集本周数据，Claude 生成 200 字犀利周报，出现在对话中
- 导出知识库为 Markdown（`## 标题 + 摘要 + 标签 + 笔记`）

### 内容获取
- RSS 订阅：支持 RSS 2.0 / Atom，批量刷新导入信息流
- Cloudflare Worker 代理：解决微信等网站 CORS 问题，Settings 中配置

### 设置页
- API Key 管理（运行时覆盖 .env）
- CORS 代理地址配置（Cloudflare Worker）
- 学习目标设置
- RSS 订阅源管理

---

## ⚠️ 已知问题

- 微信公众号文章需要代理才能直接粘贴 URL（已提供 Worker 代码，用户需自行部署）
- `parseContent` 中 URL 方式依赖网页能被 fetch 到，部分网站即使有代理也可能有反爬措施
- 手机端布局未适配（三栏在小屏会被压缩）
- 周报对话次数统计是全量而非本周增量（messages 无时间戳）
