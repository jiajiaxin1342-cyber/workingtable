## 2. 技术栈

| 层 | 技术 |
|---|---|
| 前端框架 | React 18 + Vite |
| 样式 | 手写 CSS（无 Tailwind） |
| 状态管理 | Zustand + persist 中间件 |
| AI | Anthropic Claude API（浏览器直连，流式 SSE） |
| 数据持久化 | localStorage（主） / Notion API（预留，默认关闭） |
| 后端 | 无（纯前端 SPA） |
| 部署辅助 | Cloudflare Worker（CORS 代理，手动部署） |

---

## 3. 页面 / 场景

无路由库，用 Zustand `activeScene` 状态切换场景：

```
知识库 (knowledge)
  ├── 今日信息流        ← RSS 文章
  ├── 知识库
  │   ├── 精选信息      ← 手动添加文章
  │   ├── 长期洞见      ← 写作台草稿（已发布/草稿）
  │   └── 短期观察      ← 写作台草稿（已发布/草稿）
  └── 学习进度
  + [子页面] 文章全文阅读 / WebView iframe

产品工坊 (product)
  ├── 产品观（POV）     ← 一句话判断 + 优劣 + 我来做
  ├── 我的文档          ← 写作台产品草稿 + 上传文档
  └── 外部评测          ← 社交媒体评论

设置 (settings)
  ├── Claude API Key
  ├── CORS 代理地址
  ├── 学习目标
  └── RSS 订阅源管理
```

---

## 4. 数据存储

- **主存储**：`localStorage`，key = `ai-workbench-storage`，Zustand persist 自动序列化
- **预留同步**：Notion API（`src/storage/notion.js`，`USE_NOTION = false` 关闭）
- **不持久化的运行时状态**：`writingPanelOpen`、`writingActiveTab`、`readingArticleId`、`webViewUrl`、`editingDraftId`、`pendingMentorMessage`

## 5. 核心数据模型

### 产品工坊

```js
Product {
  id: number           // Date.now()
  name: string
  createdAt: ISO string
  dimensions: Dimension[]
  pov: POV
}

Dimension {
  id: number
  name: string
  level: 0-5           // 0=未评, 1=明显短板, …, 5=行业领先
  desc: string
}

POV {
  verdict: string      // 一句话判断
  pros: string[3]      // 做对了
  cons: string[3]      // 做错了
  myTake: string       // 如果是我
}

Review {
  id, productId, type: 'social'|'mine'
  source, title, summary, content?
  savedAt: ISO string
}
```

### 知识库

```js
Article {
  id, title, source, url
  content: string      // 全文（可为空）
  summary: string
  tags: string[]
  savedAt: ISO string
  isInKB: boolean
  feedSource: 'rss'|'manual'
  linkOnly?: boolean   // 仅存链接，无内容
}

KBItem {
  id, articleId
  tags: string[]
  note: string         // 洞见（必填）
  savedAt: ISO string
}

Draft {
  id, title, body
  createdAt, updatedAt: ISO string
  status?: 'draft'|'published'
  publishedAt?: ISO string
}
// drafts: { product: Draft[], insight: Draft[], observation: Draft[] }

RssFeed {
  id, url, name
  lastFetched?: ISO string
}
```
